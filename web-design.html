CodePlay is a coding introduction application that utilizes gamification methods for children from elementary school to junior high school. In this application, you can learn the basics of various programming languages such as Python, Java, and C/C++. Of course, the material is presented in a fun and enjoyable way. You will also receive practice exercises arranged according to the teaching materials. Don't worry, if you find it challenging, you can use the hints provided. We have also designed the CodePlay interface to resemble tools found in the real world, such as terminal views and errors that you may encounter in practice exercises.

Problems

  • Background of the landing page is too crowded, so it can reduce user visibility
  • Contrasting color combinations and inconsistent font sizes
  • The store address is too full, thus allowing the main information not to be conveyed properly
  • The products sold are not accompanied by photos, this is can confuse users

Solutions

  • Redesign the website
  • Increase user visibility by changing the background on the landing page website
  • Using one type of font with size varies according to its function to make it easier for users to remember information which are given
  • Changing the color of the CTA (Call to Action) so that it doesn't contrast with the other colors
  • Add a header to put the logo and navigation menu
  • Add a photo from each product offered for anticipating users who don't know the names of these types of products
  • I also add a footer that contains the address of the main office in several regions and social media from the store it
